# VISTA M-CRITERIA ANCHOR PATTERNS

all [M] criteria use ONE of 6 reusable patterns. 1-5 scale, zero-anchored.
researcher matches evidence to level, reads off number. no vibe scoring.
each pattern has fixed sample/denominator so "some"/"most" not subjective.

## Pattern A, Reach/Presence spread
how many channels/engines show the thing.
1 = absent everywhere
2 = one
3 = some
4 = most
5 = all
denominator = defined channel/engine set (state it).

## Pattern B, Favorability
when present, how positive.
1 = only appears negative/caveated
2 = mostly caveated
3 = neutral
4 = mostly favorable
5 = consistently favorable
sample = all appearances in defined set.

## Pattern C, Earned vs supplied
original/independent vs borrowed/paid.
1 = entirely supplied/paid/stock
2 = mostly supplied
3 = mixed
4 = mostly earned/original
5 = entirely earned/original

## Pattern D, Coverage completeness
how much of a defined set is present.
1 = none of set
2 = ~quarter
3 = ~half
4 = most
5 = all
state the set.

## Pattern E, Consistency
coherence across a sample.
1 = no consistency / absent
2 = erratic
3 = loosely consistent
4 = mostly consistent, minor lapses
5 = fully consistent
sample = 20 items (posts) unless stated.

## Pattern F, Positioning fit
does execution match the tier claimed.
1 = contradicts positioning
2 = below tier
3 = adequate/generic
4 = matches tier
5 = embodies + elevates tier

## 2 bespoke scales (not patterns)

### DE23, path awareness→action
1 = only "register interest and wait", no self-guided path
2 = can browse models, no clear next step
3 = can browse + contact, journey not signposted
4 = clear path (browse→compare→book), minor gaps
5 = fully guided journey, every stage signposted

### RC8, advocacy vs silence
1 = near silence, scored neutral-default, FLAG as gap not strength
2 = sparse mentions, no genuine advocacy
3 = some discussion, mostly transactional
4 = visible advocacy from real users
5 = active self-sustaining positive community

## anchor rule
anchors must be OBSERVABLE not evaluative.
banned: "5 = excellent branding"
allowed: "5 = consistent palette+type+original photography across all 20 posts"

## inverted patterns ([M-*-inv])
formal rule for any [M-*-inv] tag (e.g. [M-B-inv], [M-D-inv]):
an inverted pattern REVERSES its base pattern's semantic level order. base level 5 meaning becomes inverted level 1; base level 1 meaning becomes inverted level 5.
the 5-end MUST always be the good/desirable state and the 1-end the deficient state. this preserves zero-anchoring (score 1 = 0 pts = worst).
scoring math is unchanged: pts = ((score-1)/4) * criterion_value.
inline anchors previously written on -inv criteria are superseded by this formal rule.
applies uniformly to: AV10 [M-B-inv], BA26 [M-B-inv], RC07 [M-B-inv], SC14 [M-D-inv].
